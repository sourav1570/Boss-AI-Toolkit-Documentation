# Getting Started

Welcome to **Boss AI Toolkit**!

Thank you for choosing Boss AI Toolkit. This toolkit has been designed to help you create engaging, high-quality boss enemies for Unity without spending weeks building AI systems from scratch.

Whether you're creating a mobile game, an action RPG, a survival game, or any project that requires intelligent boss encounters, Boss AI Toolkit provides a modular and easy-to-use framework that lets you focus on designing gameplay instead of writing complex AI code.

---

## What is Boss AI Toolkit?

Boss AI Toolkit is a complete boss AI solution for Unity that combines powerful features with a simple workflow. Every system has been built to be modular, allowing you to enable only the features your project needs.

The toolkit includes systems such as:

* Modular Boss AI Architecture
* Patrol, Wander, and Idle Behaviours
* Vision & Field of View Detection
* Automatic Target Detection
* Combat and Multi-Phase Boss Logic
* Melee and Special Attack Support
* Flying Enemy Support
* Health System with Dual Health Bars
* Ragdoll Death System
* Spine IK Target Tracking
* Event-Based Gameplay Framework
* Mobile-Ready Demo Character and Controls
* Ready-to-Use Example Bosses

Everything is designed to work together while remaining easy to customize and extend.

---

## Designed for Beginners and Professionals

Boss AI Toolkit is suitable for developers of all experience levels.

If you're new to Unity AI, the included documentation and video tutorials will guide you step by step through the setup process.

If you're an experienced developer, you'll find a clean, modular architecture that can easily be integrated into your existing project and expanded with your own custom behaviours.

---

## Before You Begin

Before importing the toolkit, it is recommended that you:

* Use a supported Unity version listed on the Asset Store.
* Create a backup of your project.
* Import the toolkit into a clean project first if you want to explore all of the included demo content.
* Read the documentation in order, starting from this Getting Started guide.
